Definition

template<typename Agvb>
class CGAL::Apollonius_graph_hierarchy_vertex_base_2< Agvb >

The class Apollonius_graph_hierarchy_vertex_base_2 provides a model for the ApolloniusGraphHierarchyVertexBase_2 concept, which is the vertex base required by the Apollonius_graph_hierarchy_2<Gt,Agds> class. The class Apollonius_graph_hierarchy_vertex_base_2 is templated by a class Agvb which must be a model of the ApolloniusGraphVertexBase_2 concept.
