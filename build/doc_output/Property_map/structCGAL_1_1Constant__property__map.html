#include <CGAL/property_map.h>

Definition

template<class KeyType, class ValueType>
struct CGAL::Constant_property_map< KeyType, ValueType >

Property map that returns a fixed value. Note that this value is chosen when the map is constructed and cannot be changed afterwards. Specifically, the free function put() does nothing.
