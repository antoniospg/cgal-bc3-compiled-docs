CGAL::Color Class Reference
IO Streams Reference

#include <CGAL/IO/Color.h>

Definition

An object of the class Color is a color available for drawing operations in many CGAL output streams. Each color is defined by a 4 unsigned chars (r,g,b,a) with 0 \(\le\) r,g,b,a \( \le \) 255, the so-called rgba-value of the color.

The alpha parameter (representing transparency) is often ignored and left to its default value (255 = no transparency), which is why we often refer to the rgb-value of the color.
