Definition

template<typename Info, typename Traits, typename Vb>
class CGAL::Triangulation_vertex_base_with_info_3< Info, Traits, Vb >

The class Triangulation_vertex_base_with_info_3 is a model of the concept TriangulationVertexBase_3, the base vertex of a 3D-triangulation. It provides an easy way to add some user defined information in vertices. Note that input/output operators discard this additional information.

Template Parameters
Infois the information the user would like to add to a vertex. It has to be DefaultConstructible and Assignable.
Traitsis the geometric traits class and must be a model of TriangulationTraits_3.
Vbis a vertex base class from which Triangulation_vertex_base_with_info_3 derives. It must be a model of the TriangulationVertexBase_3 concept. It has the default value Triangulation_vertex_base_3<Traits>.
