#include <CGAL/Fourtuple.h>

Definition

template<typename T>
class CGAL::Fourtuple< T >

The Fourtuple class stores a homogeneous (same type) fourtuple of objects of type T. A Fourtuple is much like a container, in that it "owns" its elements. It is not actually a model of container, though, because it does not support the standard methods (such as iterators) for accessing the elements of a container.

Deprecated:
This class is deprecated, and will be removed in some future CGAL release. Please use std::array instead.
