Definition

template<typename K>
class CGAL::Qt::PainterOstream< K >

An object of type PainterOstream provides output operators for Kernel objects. As CGAL has unbounded objects the PainterOstream must be constructed with a clipping rectangle. This is typically the visible area of the widget where the unbounded object is displayed.
